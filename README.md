## Introduction

*Gerrit Docker* is the Dockerfile for Gerrit Docker built on Ubuntu.



### Build

```bash
./build.sh
```



### Deploy

```bash
docker pull craftslab/gerritdocker:latest
```



### Run

```bash
./run.sh
```
